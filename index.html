Portfolio

Project 1 - Portfolio Website

(Based off of an online template) A responsive, professionally designed portfolio website showcasing my projects, skills, and professional background. This project helped me learn HTML, CSS, and web deployment, while providing an interactive platform for potential employers and collaborators to learn about my work and contact me directly.

Project 2 - Tic-Tac-Toe Game

A console-based implementation of the classic Tic-Tac-Toe game, developed in C++. This project showcases my understanding of fundamental programming concepts, including game logic and user input handling The game features a two-player mode where players take turns marking X's and O's on a 3x3 grid.

Project 3 - Inventory Management System

A robust web application leveraging Firebase for backend operations, demonstrating full CRUD (Create, Read, Update, Delete) functionality. This project highlights my ability to develop scalable, real-time database solutions for efficient inventory tracking and management in various business contexts.

Project 4 - AI-Powered Flashcard Generator (SaaS)

An innovative Software as a Service (SaaS) application that harnesses the power of Google's Gemini API to automatically generate study flashcards. Users can input custom prompts, and the AI generates tailored questions and answers, streamlining the learning process across various subjects. This project showcases my experience in AI integration, API handling, and creating practical educational tools.

Project 5 - Yapper Social Media Platform

A full-featured social media web application built with MongoDB, demonstrating my proficiency in developing complex, interactive platforms. Yapper incorporates essential social networking functionalities, including user authentication, post creation and sharing, engagement features (likes and comments), and user follow systems. This project highlights my ability to create user-centric web applications with robust backend infrastructure.
